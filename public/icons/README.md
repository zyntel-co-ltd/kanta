# PWA Icons

Add these files for full PWA install support:

- `icon-192.png` — 192×192 px
- `icon-512.png` — 512×512 px

Use a tool like [PWA Asset Generator](https://github.com/nicholasgriffintn/pwa-asset-generator) or [RealFaviconGenerator](https://realfavicongenerator.net/) to generate from your logo.
